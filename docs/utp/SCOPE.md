# Scope and Non-goals

## Scope
- Add a portable UTP package with strict typed models and CLI.
- Add deterministic run artifact logging and adapter abstraction.

## Non-goals
- Full LLM provider integration in this scaffold.
- Automatic code patch application by auditor.

## Entrypoints
- `pnpm utp:run`
- `pnpm utp:audit`
- `pnpm utp:replay`
